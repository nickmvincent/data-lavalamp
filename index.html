What This Shows

Pre-training streams are raw-ish text a model might learn broad patterns from: web pages, educational pages, PDFs, code, and other large corpora. Post-training streams are more intentional: prompts, assistant replies, ratings, preference labels, and other human feedback used to shape behavior after the base model exists.
